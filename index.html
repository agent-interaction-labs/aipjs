What aixa.js does

Three layers that turn any website into an agent-friendly platform — without rewriting your backend.

Auto-Inference Engine

Scans semantic HTML elements — search inputs, filter dropdowns, action buttons — and generates clean JSON-RPC tool schemas. Zero configuration for standard sites.

HITL Security

Every tool is classified SAFE or HIGH_RISK. Mutations like "add to cart" freeze the page and show a modal — human must approve. Prompt injection sanitization built in.

SPA-Ready Developer API

React, Vue, Next.js, Svelte — register custom tools with TypeScript types. MutationObserver watches DOM changes. Route-aware tool refresh for single-page apps.

How it works

Three steps to agent compatibility

01

Add the script tag

Drop aixa.js into your page. The auto-inference engine scans your DOM for semantic HTML and builds tool schemas on the fly.

02

Agents discover capabilities

Browser AI agents request what your site can do. aixa.js responds with structured JSON-RPC tools — search, filter, sort, actions.

03

Safe interaction, human approval

Read-only tools execute automatically. Mutations trigger a HITL modal — the user sees exactly what the agent wants to do and approves or denies.
