aip.js Documentation

The Agent Interaction Protocol. aip.js auto-infers structured JSON-RPC tool schemas from your semantic HTML, lets you control exactly what agents can access, and enforces human approval for high-risk actions — before the payload ever touches your backend.

What is the Agent Interaction Protocol?

Agent Interaction Protocol (AIP, formerly Agentic Engine Optimization) is to AI agents what SEO is to search crawlers. Just as SEO made websites readable to Googlebot, AIP makes them readable to AI agents — with a critical difference: you stay in control.

Without AIP, agents scrape your raw HTML, guess what your forms do, and might trigger actions you never intended. With aip.js, you broadcast structured capabilities that agents consume safely — and you decide what's exposed, what's read-only, and what requires explicit human approval.

Key Concepts

🔍

Auto-Inference

Scans your semantic HTML and auto-generates JSON-RPC tool schemas. Zero config — works with the markup you already have.
⚙️

Manual Config

Full developer API for complex SPAs. Register custom search endpoints, actions, and tools with fine-grained control.
🛡️

HITL Security

Every tool classified SAFE or HIGH_RISK. Mutations freeze the page and require human approval — enforced, not suggested.
📦

Plugins

Ready-made plugins for e-commerce, real estate, and travel. Or build your own. All read-only by design.
