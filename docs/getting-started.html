Getting Started

aip.js is a lightweight SDK that exposes your web application's functionality to AI agents via JSON-RPC tool schemas. This guide walks you through installation, quick-start examples, and verification — so you can go from zero to agent-friendly in under a minute.

← Previous: Introduction Next: Auto-Inference →

Installation

Choose the method that best fits your project:

1. npm

npm install @aipjs/core

Then import it in your application:

import { AIP } from '@aipjs/core';

2. CDN — jsDelivr

<script type="module">
  import { AIP } from 'https://cdn.jsdelivr.net/npm/@aipjs/core/dist/index.js';
</script>

3. CDN — unpkg

<script type="module">
  import { AIP } from 'https://unpkg.com/@aipjs/core/dist/index.js';
</script>

4. CDN — esm.sh (bundled)

<script type="module">
  import { AIP } from 'https://esm.sh/@aipjs/core';
</script>

5. Self-Hosted Bundle

Download the bundle entry point from demo/vendor/bundle-entry.mjs, host it on your own server, and import it directly: 

<script type="module">
  import { AIP } from '/path/to/your/bundle-entry.mjs';
</script>

Quick Start — Auto-Inference

Auto-inference scans your page's semantic HTML and automatically generates JSON-RPC tool schemas. No manual configuration is needed — ideal for server-rendered pages and simple SPAs. 

import { AIP } from '@aipjs/core';

const aip = new AIP({ autoInfer: true });
aip.start();

That's it. The SDK will:

  • Scan the DOM for semantic elements (search inputs, forms, buttons, links, etc.)
  • Generate tool schemas with names, descriptions, and parameter definitions
  • Broadcast available tools via CustomEvent on window
  • Listen for incoming JSON-RPC requests from AI agents

Quick Start — Manual Config

For complex SPAs, custom APIs, or when you need fine-grained control over what agents can access, use the manual registration API: 

import { AIP } from '@aipjs/core';

const aip = new AIP();

aip.registerSearch('product_search', [
  { name: 'q', type: 'string', description: 'Search query', required: true },
  { name: 'category', type: 'string', description: 'Product category' },
  { name: 'minPrice', type: 'number', description: 'Minimum price' },
  { name: 'maxPrice', type: 'number', description: 'Maximum price' }
], 'https://api.example.com/products/search');

aip.start();

Verifying It Works

Once the SDK is running, open your browser's DevTools console and run: 

window.__aip.getCapabilities()

This returns an array of all registered tool schemas — both auto-inferred and manually registered. Each entry includes the tool's name, description, parameters schema, riskLevel, and category.

Tip: You can also listen for the aip:tools-ready event to know when tool discovery is complete: 
window.addEventListener('aip:tools-ready', (e) => {
  console.log('Available tools:', e.detail.tools);
});
